# 🌦️ HackStack Weather App

A responsive weather application built using HTML, CSS, and JavaScript that provides real-time weather information for cities worldwide.

## Features
- Real-time weather updates
- City-based weather search
- Temperature, humidity, and wind details
- Responsive and user-friendly interface
- Dynamic weather condition icons

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API
